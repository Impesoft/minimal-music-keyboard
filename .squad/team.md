# Squad Team

## Project Context

**Project:** Minimal Music Keyboard — lightweight WinUI3 MIDI player for Windows 11
**Stack:** WinUI3 (Windows App SDK), C#/.NET
**What it does:** System-tray resident MIDI player. Listens to a configured MIDI device (Arturia KeyLab 88 MkII), routes MIDI input to selected software instruments (soundfonts/synthesis). Users switch instruments via MIDI commands. Settings GUI, zero memory leaks, lives in the taskbar notification area.
**User:** Ward Impe
