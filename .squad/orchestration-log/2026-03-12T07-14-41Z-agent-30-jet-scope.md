## Recommendation Details
- Bridge creates independent Win32 popup (`CreateWindowExW WS_POPUP`)
- Calls `IPlugView::attached()` with bridge HWND
- Runs dedicated Win32 message pump thread (isolated from audio)
- IPC commands: `openEditor`, `closeEditor`
- IPC events: `editorOpened` (with dimensions), `editorClosed`
- C# adds "Open Editor" button in VST3 settings row

## Risk & Mitigation
- **Plugin GUI compatibility:** Not all VST3 instruments implement `IEditController`; error path required
- **Threading:** Editor message pump must not block audio render thread (separate thread)
- **Stability:** Matches standard DAW out-of-process plugin scanning model
